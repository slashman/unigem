# UniGEM
A collection of Unity UI elements to mimick DOS GEM UI

![Sample Scene](screenshot.png?raw=true "Sample Scene")

## Authors
Santiago Zapata

## How to use it
- Include the `UniGEM` directory inside the Assets directory of your Unity project
- Create instances of the Prefabs to replicate the widgets.
